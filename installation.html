Installationïƒ

Warning

Python Requirements

EMF CLI requires Python version 3.10 or above. You also need to add it to the PATH.

Install EMF Command Line Interfaceïƒ

Install manuallyïƒ

To download the CLI manually, go to EMF CLI Releases . Scroll down to the Assets section of the page and click the link (.gz or .zip) for your OS and extract the files. Make sure your system Path includes the path to the EMF CLI binary you downloaded.

Note: On macOS, if youâ€™re not able to run commands, you may need to change the binaryâ€™s permissions to allow execution. From the binaryâ€™s folder, run: chmod 755 emf-cli.

Install using Goïƒ

Note

Go Requirements

If you wish to install through Go, then please make sure you are using version 1.21 or above.

Install it directly using the following commandïƒ

go install github.com/easy-model-fusion/emf-cli@latest

Install it from sourceïƒ

First get the source code from the repository :

git clone https://github.com/easy-model-fusion/emf-cli
cd emf-cli/

Then you can start building (using make) :

make build

Or you could also build it yourself :

go build

Now all thatâ€™s left for you is adding it to the PATH !

Verify installationïƒ

After installation, you will have access to the emf-cli binary in your command line.

You can check you have the right version with this command:

emf-cli version
